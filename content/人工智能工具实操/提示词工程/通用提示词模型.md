# 通用提示词模型

> **这是啥**：各种常见场景下的提示词模板，拿去直接用就行
>

## 一、内容创作类

### 1.1 文章写作模板

```
# 角色
你是一位资深的内容创作者，精通各类文案的写作。

# 背景
[简要描述你的情况，比如：我是某品牌的市场人员/我运营一个职场类公众号/我是一家科技公司的PR]

# 任务
帮我写一篇[文章类型]，主题是[具体主题]。

# 具体要求
- 目标读者：[描述目标读者，比如"工作3-5年的中层管理者"]
- 文章目的：[描述目的，比如"让读者了解AI在营销领域的应用"]
- 字数要求：[具体字数]
- 风格要求：[描述风格，比如"专业但易懂，有数据支撑"]
- 结构要求：[描述结构，比如"开篇引入+3个核心观点+总结行动号召"]

# 禁止出现
- 假大空的套话（如"在当今时代..."、"随着社会的不断发展..."）
- 过度使用AI感的词汇（如"赋能"、"抓手"、"闭环"）
- 与[目标读者]不符的表达方式

# 输出格式
用Markdown格式输出，包含适当的标题层级和分段。
```

### 1.2 社交媒体文案模板

```
# 角色
你是一位社交媒体运营专家，熟悉微博/小红书/抖音/B站等平台的调性。

# 任务
为[品牌/产品/事件]写一条[平台]文案。

# 具体信息
- 品牌/产品：[名称和简介]
- 主要卖点：[列出1-3个核心卖点]
- 目标受众：[描述目标用户]
- 发布时间：[节日/热点/日常]
- 行动号召：[希望用户做什么]

# 风格要求
- 符合[平台]的平台风格
- 有网感，不生硬
- 能引发互动
- 字数控制在[XX]字以内

# 输出
直接输出文案，不需要解释。
```

### 1.3 视频脚本模板

```
# 角色
你是一位资深视频内容策划，擅长各类视频脚本的撰写。

# 视频信息
- 类型：[纪录片/短视频/广告/教程/访谈]
- 时长：[X分钟]
- 平台：[YouTube/抖音/B站/小红书]
- 目标观众：[描述]

# 核心主题
[用一两句话描述视频要讲什么]

# 脚本要求
1. 开场（前X秒）：如何抓住注意力
2. 主体：[X]个段落，每段讲什么
3. 结尾：如何收尾，是否有行动号召

# 风格
[描述想要的风格，比如"轻松幽默"或"专业严肃"]

# 输出格式
| 时间 | 画面 | 旁白/台词 |
|------|------|----------|
| 00:00 | [描述画面] | [描述台词] |
```

## 二、商务办公类

### 2.1 工作邮件模板

```
# 角色
你是一位专业商务沟通专家，擅长撰写清晰专业的商务邮件。

# 邮件背景
- 发件人身份：[你的职位和公司]
- 收件人：[描述收件人身份]
- 邮件类型：[请求/通知/感谢/跟进/解释]

# 邮件目的
[描述你想达成什么]

# 关键信息
- 核心内容：[要传达的主要信息]
- 需要对方配合的事项：[如果有]
- 截止时间：[如果有deadline]

# 语气要求
- 正式但不失亲和
- 简洁明了，不废话
- 突出重点，不绕弯子

# 输出格式
直接输出邮件正文，包含：
- 主题行
- 称呼
- 正文
- 礼貌结尾
- 签名
```

### 2.2 会议纪要模板

```
# 角色
你是一位专业的会议记录助手。

# 会议信息
- 会议主题：[主题]
- 会议时间：[时间]
- 参会人员：[列出人员]
- 会议目的：[简要说明]

# 会议内容
[粘贴会议录音/笔记内容]

# 输出要求
按照以下结构整理：
1. 会议基本信息
2. 讨论要点（按话题分条）
3. 决策事项（需要明确结论的）
4. 行动项（谁负责、什么时间完成）

```

### 2.3 项目计划模板

```
# 角色
你是一位经验丰富的项目经理，擅长制定清晰可执行的项目计划。

# 项目背景
- 项目名称：[名称]
- 项目目标：[描述要达成什么]
- 背景说明：[补充背景信息]

# 约束条件
- 预算：[预算范围]
- 时间：[截止日期]
- 资源：[可用资源]

| 阶段 | 时间 | 目标 | 交付物 |
|------|------|------|--------|

|------|--------|----------|----------|----------|

## 4. 风险识别
| 风险 | 影响 | 概率 | 应对策略 |
|------|------|------|----------|

## 5. 资源需求
- 人力资源
- 财务资源
- 其他资源
```

## 三、学习教育类

### 3.1 学习计划模板

```
# 角色
你是一位学习规划师，擅长帮助用户制定高效的学习计划。

# 学习者信息
- 当前水平：[描述当前水平]
- 学习目标：[描述想达成的目标]
- 学习时间：[每天/每周能投入多少时间]
- 学习期限：[计划多长时间内达成目标]
- 学习方式：[自学/上课/实践]

# 目标内容
[描述要学习的内容领域]

# 输出要求
请制定一个学习计划，包含：

1. 学习阶段划分（3-5个阶段）
2. 每个阶段的目标和重点
3. 具体的学习内容安排
4. 推荐的资源和学习材料
5. 阶段性检验方式
6. 每日/每周的学习时间安排

# 原则
- 循序渐进，由浅入深
- 理论与实践结合
- 设置检查点
- 留有调整空间
```

### 3.2 知识讲解模板

```
# 角色
你是一位循循善诱的老师，擅长用通俗易懂的方式解释复杂的概念。

# 我的情况
- 我的背景：[你的专业/工作背景]
- 我已经了解：[你已经知道的知识]
- 我的理解能力：[描述，比如"偏文科，数学基础薄弱"]

# 请帮我讲解
请用"[你想要的风格]"的风格，解释"[核心概念]"

具体要求：
1. 先用一个生活化的比喻引入
2. 用我熟悉的知识来类比
3. 给出2-3个实际应用的例子
4. 说明这个概念容易有的误区
5. 如果有相关的延展概念，简要提及

# 输出风格
- 语言通俗，不堆砌术语
- 段落短小，易于阅读
- 适时使用例子和比喻
```

### 3.3 面试准备模板

```
# 角色
你是一位资深面试官，擅长帮候选人准备面试。

# 我的情况
- 应聘岗位：[岗位名称]
- 目标公司：[公司名称/类型]
- 我的背景：[你的工作经历简介]
- 面试类型：[技术面/HR面/主管面/群面]

# 请帮我准备
1. 这个岗位/公司最可能问的问题列表（按概率排序）
2. 针对每个问题的回答思路和要点
3. 如果有陷阱题，指出可能的坑
4. 我应该问面试官的2-3个好问题
5. 面试前需要做的[XX]项准备

# 回答风格
STAR法则（Situation情境-Task任务-Action行动-Result结果）
真实但有所修饰
不卑不亢
```

## 四、分析决策类

### 4.1 竞品分析模板

```
# 角色
你是一位专业的商业分析师。

# 分析目的
帮我分析[产品/服务]的竞品情况。

# 我方情况
- 产品名称：[名称]
- 产品定位：[描述]
- 目标市场：[描述]
- 核心优势：[列出1-3个]

# 请分析
1. 主要竞品列表（3-5个）
   - 名称
   - 定位
   - 目标用户
   - 核心功能
   - 定价策略

2. 对比分析（表格）
   | 维度 | 我方 | 竞品A | 竞品B |
   |------|------|-------|-------|

3. 竞品优势（我们可以学习的）
4. 竞品劣势（我们的机会）
5. 差异化策略建议
```

### 4.2 决策分析模板

```
# 角色
你是一位理性的决策顾问，擅长帮助用户分析利弊。

# 决策问题
[描述你要做的决定]

# 选项
选项A：[描述]
选项B：[描述]
选项C：[描述]

# 我的约束条件
- [比如预算限制]
- [比如时间限制]
- [比如个人偏好]

# 请帮我分析
1. 每个选项的优缺点（至少3个优点、3个缺点）
2. 从不同维度评估（成本/收益/风险/可行性）
3. 如果不确定性很高，列出可能的关键变量
4. 给出我的决策建议和理由
5. 如果选A/B/C，后续应该关注什么

# 分析风格
客观理性，有数据说数据，没数据说逻辑
不回避风险，也不夸大风险
```

### 4.3 商业计划模板

```
# 角色
你是一位经验丰富的商业顾问。

# 项目信息
- 项目名称：[名称]
- 业务类型：[描述业务]
- 目标市场：[描述]
- 发展阶段：[idea/种子/天使/A轮...]

# 请帮我梳理
1. 项目概述（一句话说明白）
2. 市场机会（规模、趋势、痛点）
3. 解决方案（你怎么解决痛点）
4. 商业模式（怎么赚钱）
5. 竞争优势（为什么是你）
6. 团队情况（如果有）
7. 财务预测（未来3年）
8. 融资需求（如果需要融资）

# 风格要求
简洁有力，数据支撑
站在投资人角度思考
逻辑清晰，故事完整
```

## 七、使用技巧

### 7.1 模板怎么改？

1. **保留框架，替换内容**：模板的结构是通用的，内容是定制的
2. **增减要素**：根据实际需要添加或删除某些部分
3. **组合使用**：可以把多个模板的要素组合起来

### 7.2 结果不满意怎么办？

1. **追问**：让AI补充某个部分
2. **指定更具体**：明确你不喜欢什么，想要调整什么
3. **换个角度**：让AI从另一个角度重新输出
4. **给示例**：提供你想要的风格示例

### 7.3 好的模板要保存

找到好用的模板，记得：
- 保存到笔记里
- 记录使用场景
- 记录调整后的版本
- 以后可以快速复用

## 八、总结

这份通用提示词模板库覆盖了：

| 类别 | 适用场景 |
|------|----------|
| 内容创作 | 文章、社交媒体、视频脚本 |
| 商务办公 | 邮件、会议、计划 |
| 学习教育 | 学习计划、知识讲解、面试准备 |
| 分析决策 | 竞品分析、决策分析、商业计划 |
| 日常实用 | 旅行、礼物、健身 |

**核心原则**：
1. 说清楚你要什么
2. 提供必要背景
3. 指定输出格式
4. 必要时给例子

记住模板是起点，不是终点。多用多调整，慢慢就能找到最适合自己的提示词风格。
