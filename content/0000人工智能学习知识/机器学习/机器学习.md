#### 1.1.1 逻辑回归详解

逻辑回归虽然名字带"回归"，实际是分类算法。核心公式：

$$
P(y=1|x) = \sigma(w^Tx + b) = \frac{1}{1 + e^{-(w^Tx + b)}}
$$

```python
import numpy as np

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

class LogisticRegression:
    def __init__(self, lr=0.01, n_iters=1000):
        self.lr = lr
        self.n_iters = n_iters
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0
        
        # 梯度下降
        for _ in range(self.n_iters):
            linear = np.dot(X, self.weights) + self.bias
            predictions = sigmoid(linear)
            
            dw = (1/n_samples) * np.dot(X.T, (predictions - y))
            db = (1/n_samples) * np.sum(predictions - y)
            
            self.weights -= self.lr * dw
            self.bias -= self.lr * db
    
    def predict(self, X):
        linear = np.dot(X, self.weights) + self.bias
        return [1 if p > 0.5 else 0 for p in sigmoid(linear)]
```

$$

$$

#### 1.2.1 线性回归详解

损失函数为均方误差（MSE）：

$$
\mathcal{L} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2
$$

闭式解：
$$
w = (X^TX)^{-1}X^Ty
$$

```python
import numpy as np

class LinearRegression:
    def __init__(self, lr=0.01, n_iters=1000):
        self.lr = lr
        self.n_iters = n_iters
        self.weights = None
        self.bias = None
    
    def fit(self, X, y):
        n_samples, n_features = X.shape
        self.weights = np.zeros(n_features)
        self.bias = 0
        
        for _ in range(self.n_iters):
            y_pred = np.dot(X, self.weights) + self.bias
            
            dw = (1/n_samples) * np.dot(X.T, (y_pred - y))
            db = (1/n_samples) * np.sum(y_pred - y)
            
            self.weights -= self.lr * dw
            self.bias -= self.lr * db
    
    def predict(self, X):
        return np.dot(X, self.weights) + self.bias
```

#### 2.1.1 K-Means算法

```python
import numpy as np

def kmeans(X, k, max_iters=100):
    """K-Means聚类实现"""
    n_samples, n_features = X.shape
    
    # 随机初始化质心
    centroids = X[np.random.choice(n_samples, k, replace=False)]
    
    for _ in range(max_iters):
        # 分配样本到最近质心
        distances = np.sqrt(((X - centroids[:, np.newaxis])**2).sum(axis=2))
        labels = np.argmin(distances, axis=0)
        
        # 更新质心
        new_centroids = np.array([X[labels == i].mean(axis=0) for i in range(k)])
        
        # 检查收敛
        if np.allclose(centroids, new_centroids):
            break
        centroids = new_centroids
    
    return centroids, labels

# sklearn实现
from sklearn.cluster import KMeans
kmeans = KMeans(n_clusters=3, random_state=42)
labels = kmeans.fit_predict(X)
```

## 三、半监督学习（Semi-Supervised Learning）

> [!tip]
> 半监督学习介于监督和无监督之间，利用少量标注数据和大量未标注数据提升模型性能。

### 3.1 常见方法

| 方法 | 核心思想 | 实现方式 |
|------|----------|----------|
| **自训练** | 用标注数据训练，预测未标注数据，加入高置信样本 | 迭代伪标签 |
| **协同训练** | 多视图数据，两个分类器互相教学 | 多模型互补 |
| **标签传播** | 图-based，近邻样本标签平滑 | 标签扩散 |
| **半监督SVM** | 考虑未标注数据寻找最大间隔 | S3VM |

### 3.2 自训练示例

```python
def self_training(model, X_labeled, y_labeled, X_unlabeled, threshold=0.9, max_iter=10):
    """自训练伪标签方法"""
    X_pool = X_unlabeled.copy()
    
    for _ in range(max_iter):
        if len(X_pool) == 0:
            break
        
        # 用标注数据训练模型
        model.fit(X_labeled, y_labeled)
        
        # 预测未标注数据
        probs = model.predict_proba(X_pool)
        max_probs = probs.max(axis=1)
        
        # 筛选高置信样本
        confident_mask = max_probs >= threshold
        if not confident_mask.any():
            break
        
        # 加入训练集
        pseudo_labels = model.predict(X_pool[confident_mask])
        X_labeled = np.vstack([X_labeled, X_pool[confident_mask]])
        y_labeled = np.hstack([y_labeled, pseudo_labels])
        
        # 移出已标注数据
        X_pool = X_pool[~confident_mask]
    
    return model
```

|------|------|----------|

| NLP | 数据标注耗时 | 自训练、预训练 |

## 四、强化学习（Reinforcement Learning）

> [!warning]
> 强化学习通过与环境交互学习最优策略，核心是**奖励信号**和**决策序列**。

### 4.1 核心概念

| 概念 | 定义 | 数学表示 |
|------|------|----------|
| **智能体** | Agent | 学习者/决策者 |
| **环境** | Environment | 智能体所处的世界 |
| **状态** | State $s$ | 环境在时刻t的描述 |
| **动作** | Action $a$ | 智能体可采取的行动 |
| **奖励** | Reward $r$ | 环境反馈的信号 |
| **策略** | Policy $\pi$ | 状态到动作的映射 $\pi(a|s)$ |
| **价值函数** | Value Function $V(s)$ | 状态的长期价值 |
| **Q函数** | Q Function $Q(s,a)$ | 状态-动作对的价值 |

#### 4.2.2 无模型强化学习

**基于值函数（Value-Based）**

| 算法 | 核心思想 | 特点 |
|------|----------|------|
| **Q-Learning** | 时序差分更新Q表 | 离线策略 |
| **DQN** | 深度Q网络+经验回放 | 突破高维状态 |
| **Double DQN** | 解决Q值过估计 | 更稳定 |
| **Dueling DQN** | 分离V和A | 样本效率高 |
| **Rainbow** | DQN集大成者 | 7种技巧融合 |

```python
import numpy as np

class QLearningAgent:
    def __init__(self, n_states, n_actions, lr=0.1, gamma=0.95, epsilon=0.1):
        self.n_states = n_states
        self.n_actions = n_actions
        self.lr = lr
        self.gamma = gamma
        self.epsilon = epsilon
        self.q_table = np.zeros((n_states, n_actions))
    
    def choose_action(self, state):
        if np.random.uniform(0, 1) < self.epsilon:
            return np.random.randint(self.n_actions)
        return np.argmax(self.q_table[state])
    
    def learn(self, state, action, reward, next_state, done):
        current_q = self.q_table[state, action]
        if done:
            target_q = reward
        else:
            target_q = reward + self.gamma * np.max(self.q_table[next_state])
        
        self.q_table[state, action] += self.lr * (target_q - current_q)
```

## 五、模型选择指南

> [!tip] 如何选择合适的模型
> 
> 1. **数据量小**（<1万样本）：传统ML优先
>    - 树模型（XGBoost、随机森林）
>    - 逻辑回归、SVM
> 
> 2. **数据量大**（>100万）：深度学习或大规模ML
>    - 神经网络、Transformer
>    - 分布式XGBoost
> 
> 3. **特征少**（<100）：任意模型
>    - 尝试多种模型对比
> 
> 4. **特征多**（>1万）：降维+ML或深度学习
>    - PCA降维后用树模型
>    - 直接用神经网络
> 
> 5. **需要可解释**：线性模型、决策树
>    - 逻辑回归、线性回归
>    - 决策树、SHAP解释
