# 凸优化深度指南

> [!note] 文档概述
> 凸优化是现代机器学习与统计推断的理论支柱。本指南系统介绍凸集与凸函数的基本理论、典型优化问题（LP、QP）、对偶理论与KKT条件、梯度下降法及其收敛性分析，以及各类一阶二阶优化方法。

## 一、凸集与凸函数

### 1.1 凸集的定义与性质

**凸集**：集合 $C \subseteq \mathbb{R}^n$ 是凸的，当且仅当对任意 $x, y \in C$ 和 $\theta \in [0, 1]$，有：

$$\theta x + (1-\theta)y \in C$$

几何直观：凸集内任意两点的线段完全包含在集合内。

**常见的凸集**：
- **超平面**：$\{x : a^T x = b\}$
- **半空间**：$\{x : a^T x \leq b\}$
- **多面体**：有限个半空间和超平面的交集
- **范数球**：$\{x : \|x\| \leq 1\}$（任意范数）
- **仿射集**：$\{x : Ax = b\}$
- **正象限**：$\mathbb{R}_+^n$

**凸集的运算保凸性**：
- 交集：凸集的交集仍是凸集
- 和：如果 $C_1, C_2$ 凸，则 $C_1 + C_2 = \{x+y : x \in C_1, y \in C_2\}$ 凸
- 仿射变换：$f(C) = \{f(x) : x \in C\}$ 保持凸性
- 透视变换：保持凸性

> [!example] 凸集示例
> 1. 单位球 $B = \{x : \|x\|_2 \leq 1\}$ 是凸的
> 2. 三角形的内部是凸的
> 3. 圆盘（平面内）是凸的
> 4. 环形区域 $\{x : 1 \leq \|x\| \leq 2\}$ 不是凸的（因为不包含原点到边界点的线段）

### 1.2 凸函数的定义

**凸函数**：函数 $f: \mathbb{R}^n \to \mathbb{R} \cup \{+\infty\}$ 是凸的，当且仅当 $\text{dom}(f)$ 是凸集，且对任意 $x, y \in \text{dom}(f)$ 和 $\theta \in [0, 1]$：

$$f(\theta x + (1-\theta)y) \leq \theta f(x) + (1-\theta)f(y)$$

几何意义：函数图像上任意弦线位于函数图像上方。

**严格凸**：若上述不等式在 $x \neq y$ 和 $\theta \in (0,1)$ 时严格成立。

**凹函数**：$-f$ 是凸的。

**Jensen不等式**：凸函数的离散形式：对任意 $x_1, \ldots, x_k$ 和 $\theta_i \geq 0$，$\sum \theta_i = 1$：

$$f\left(\sum_i \theta_i x_i\right) \leq \sum_i \theta_i f(x_i)$$

### 1.3 凸函数的判别与性质

**一阶条件**：若 $f$ 可微，则 $f$ 凸当且仅当

$$f(y) \geq f(x) + \nabla f(x)^T (y - x), \quad \forall x, y$$

即函数位于任一点切平面的上方（支持超平面性质）。

**二阶条件**：若 $f$ 二阶可微，则 $f$ 凸当且仅当其Hessian矩阵半正定：

$$\nabla^2 f(x) \succeq 0, \quad \forall x \in \text{dom}(f)$$

**常见的凸函数**：
- 指数函数 $e^{ax}$
- 负对数 $-\log x$（在 $x > 0$）
- 范数 $\|x\|_p$（任意 $p \geq 1$）
- 二次函数 $\frac{1}{2}x^T Qx + b^T x + c$（$Q \succeq 0$）
- 最大函数 $\max(x_1, \ldots, x_n)$
- 指数和函数 $\log\sum_i e^{x_i}$（log-sum-exp）

> [!note] 凸函数的重要性
> 凸优化问题的局部最优解即全局最优解。这一性质使得凸问题在理论上"容易"求解——只需找到局部最优即可。

## 二、典型优化问题

### 2.1 线性规划（LP）

**线性规划**的标准形式：

$$\min_{x} \quad c^T x$$
$$\text{s.t.} \quad Ax = b$$
$$\quad \quad \quad x \geq 0$$

或不等式形式：

$$\min_{x} \quad c^T x$$
$$\text{s.t.} \quad Ax \leq b$$

**线性规划的几何**：目标函数定义一族平行超平面。约束定义一个多面体（凸集）。最优解出现在多面体的顶点。

**单纯形法**：通过在顶点间移动来寻找最优解，最坏情况指数时间，但实际中通常很快。

**内点法**：通过障碍函数在多面体内部移动，多项式时间复杂度。

> [!example] diet problem（饮食问题）
> 最小化成本选择食物满足营养需求：
> $$\min \sum_j c_j x_j \quad \text{s.t.} \sum_j a_{ij} x_j \geq b_i, x_j \geq 0$$
> 其中 $x_j$ 是食物 $j$ 的量，$c_j$ 是单价，$a_{ij}$ 是营养 $i$ 的含量，$b_i$ 是每日需求。

### 2.2 二次规划（QP）

**二次规划**：

$$\min_x \quad \frac{1}{2}x^T Qx + c^T x$$
$$\text{s.t.} \quad Ax \leq b$$

当 $Q \succeq 0$（半正定）时，问题为凸二次规划，全局最优可达。

**等效于**：最小二乘的正则化形式

$$\min_x \|Ax - b\|^2 + \lambda \|x\|^2$$

**KKT条件简化**（无不等式约束时）：

$$\nabla f(x) = Qx + c = 0 \Rightarrow x^* = -Q^{-1}c$$

### 2.3 二次约束二次规划（QCQP）

**QCQP**：

$$\min_x \quad \frac{1}{2}x^T Q_0 x + c_0^T x$$
$$\text{s.t.} \quad \frac{1}{2}x^T Q_i x + c_i^T x + r_i \leq 0, \quad i = 1, \ldots, m$$

当所有 $Q_i \succeq 0$ 时，QCQP是凸问题。

**二阶锥规划（SOCP）** 是QCQP的推广：

$$\min_x \quad f^T x$$
$$\text{s.t.} \quad \|A_i x + b_i\|_2 \leq c_i^T x + d_i$$

SOCP包含LP、QP和QCQP作为特例，在信号处理和鲁棒优化中广泛应用。

## 三、对偶理论与KKT条件

### 3.1 拉格朗日函数

考虑标准优化问题（**原问题**）：

$$\min_x \quad f_0(x)$$
$$\text{s.t.} \quad f_i(x) \leq 0, \quad i = 1, \ldots, m$$
$$\quad \quad \quad h_j(x) = 0, \quad j = 1, \ldots, p$$

**拉格朗日函数**：

$$L(x, \lambda, \nu) = f_0(x) + \sum_{i=1}^m \lambda_i f_i(x) + \sum_{j=1}^p \nu_j h_j(x)$$

其中 $\lambda_i \geq 0$（不等式约束的乘子），$\nu_j$（等式约束的乘子）。

### 3.2 拉格朗日对偶函数

**对偶函数**定义为拉格朗日函数关于 $x$ 的下确界：

$$g(\lambda, \nu) = \inf_{x \in \mathcal{D}} L(x, \lambda, \nu)$$

其中 $\mathcal{D} = \bigcap_i \text{dom}(f_i) \cap \bigcap_j \text{dom}(h_j)$。

**对偶函数的性质**（弱对偶性）：

$$g(\lambda, \nu) \leq p^*, \quad \forall \lambda \geq 0, \forall \nu$$

即对偶函数给出原问题最优值 $p^*$ 的下界。

### 3.3 对偶问题

**对偶问题**（拉格朗日对偶）：

$$\max_{\lambda, \nu} \quad g(\lambda, \nu)$$
$$\text{s.t.} \quad \lambda \geq 0$$

对偶问题的最优值记为 $d^*$，恒有 $d^* \leq p^*$（弱对偶性）。

**强对偶性**：在某些条件下，$d^* = p^*$。这些条件包括：

- **Slater条件**：存在严格可行的 $x$（对于凸问题），即 $\exists x$ 使得 $f_i(x) < 0$（而非 $\leq 0$）且 $Ax = b$。
- **约束规范**：如线性约束独立、KKT条件满足等。

### 3.4 KKT条件

对于最优解 $x^*$ 和对偶变量 $\lambda^*, \nu^*$，KKT条件为：

1. **原始可行性**：$f_i(x^*) \leq 0$，$h_j(x^*) = 0$
2. **对偶可行性**：$\lambda_i^* \geq 0$
3. **互补松弛**：$\lambda_i^* f_i(x^*) = 0$
4. **拉格朗日平稳性**：$\nabla f_0(x^*) + \sum_i \lambda_i^* \nabla f_i(x^*) + \sum_j \nu_j^* \nabla h_j(x^*) = 0$

> [!note] KKT的意义
> 在强对偶性成立的条件下，KKT条件是 $x^*$ 为原问题最优解的**充要条件**。这使得我们可以将约束优化问题转化为解方程组。

**几何解释**：在最优解处，梯度 $\nabla f_0(x^*)$ 可以表示为约束梯度的非负线性组合，且约束在边界处必须"夹住"梯度方向。

## 四、梯度下降与收敛性分析

### 4.1 梯度下降法

**梯度下降**是最基本的迭代优化方法：

$$x_{k+1} = x_k - \eta_k \nabla f(x_k)$$

其中 $\eta_k > 0$ 是步长（学习率）。

**收敛性分析的基本假设**：

1. **Lipschitz连续梯度**：$\|\nabla f(x) - \nabla f(y)\| \leq L \|x - y\|$，即Hessian有界：$\|\nabla^2 f(x)\| \leq L$
2. **步长选择**：固定步长 $\eta \leq \frac{1}{L}$ 或随迭代变化

### 4.2 凸函数的收敛率

**一般凸函数**（Lipschitz连续）：
$$f(x_k) - f(x^*) \leq \frac{\|x_0 - x^*\|^2}{2\eta k}$$

取最优步长 $\eta = \frac{\|x_0 - x^*\|}{L\sqrt{k}}$，得 $O(1/\sqrt{k})$ 收敛率。

**强凸函数**：设 $f$ $\mu$-强凸，即 $f(x) - \frac{\mu}{2}\|x\|^2$ 凸。

取 $\eta = 1/L$，则：
$$f(x_k) - f(x^*) \leq \left(1 - \frac{\mu}{L}\right)^k (f(x_0) - f(x^*))$$

收敛率为 $O((1 - \frac{\mu}{L})^k)$，即**线性收敛**。

> [!note] 条件数的影响
> 收敛速度依赖于**条件数** $\kappa = L/\mu$。条件数越大，收敛越慢。这解释了为什么预处理（preconditioning）可以加速优化。

### 4.3 非凸函数的收敛性

对于非凸函数，梯度下降保证：
- 梯度范数收敛到0：$\|\nabla f(x_k)\| \to 0$
- 收敛到**稳定点**（梯度为零的点），不一定是全局最优

在深度学习中，损失函数是非凸的，但梯度下降仍能有效找到好的局部最优解（实践中往往也接近全局最优）。

## 五、一阶与二阶优化方法

### 5.1 一阶方法

**动量法**（Momentum）：
$$v_{k+1} = \beta v_k + \nabla f(x_k)$$
$$x_{k+1} = x_k - \eta v_{k+1}$$

动量累积历史梯度方向，加速收敛并减少震荡。Nesterov动量更激进：

$$v_{k+1} = \beta v_k + \nabla f(x_k - \eta v_k)$$

**Adam**（Adaptive Moment Estimation）：
$$m_k = \beta_1 m_{k-1} + (1-\beta_1) \nabla f(x_k) \quad \text{（一阶矩）}$$
$$v_k = \beta_2 v_{k-1} + (1-\beta_2)(\nabla f(x_k))^2 \quad \text{（二阶矩）}$$
$$\hat{m}_k = \frac{m_k}{1-\beta_1^k}, \quad \hat{v}_k = \frac{v_k}{1-\beta_2^k}$$
$$x_{k+1} = x_k - \eta \frac{\hat{m}_k}{\sqrt{\hat{v}_k} + \epsilon}$$

Adam自动调整每个参数的学习率，在实践中效果极佳。

### 5.2 二阶方法

**牛顿法**：
$$x_{k+1} = x_k - (\nabla^2 f(x_k))^{-1} \nabla f(x_k)$$

牛顿法的收敛率：对于强凸函数，达到 $O(e^{-c k^2})$（二次收敛），远超线性收敛。

但牛顿法的问题：
1. 需要计算和存储Hessian矩阵（$O(n^2)$ 存储）
2. 需要求解Hessian线性系统（$O(n^3)$ 计算）
3. Hessian可能不正定

**拟牛顿法**：用低秩矩阵逼近Hessian或其逆：
- BFGS：逼近Hessian
- L-BFGS：限制记忆的BFGS变体
- OWL-QN：用于L1正则化问题

### 5.3 约束优化的投影方法

**投影梯度法**：
$$x_{k+1} = \text{Proj}_C\left(x_k - \eta_k \nabla f(x_k)\right)$$

其中 $\text{Proj}_C(x) = \arg\min_{y \in C} \|x - y\|$ 是到凸集 $C$ 的投影。

对于简单约束（如箱约束 $x \in [l, u]$），投影有闭式解：

$$\text{Proj}_{[l,u]}(x)_i = \min(\max(x_i, l_i), u_i)$$

## 六、Frank-Wolfe算法

### 6.1 算法描述

**Frank-Wolfe算法**（也称条件梯度法）是处理线性约束凸优化问题的投影自由方法：

**初始化**：$x_0 \in \mathcal{D}$

**迭代**：
1. **线性最小化子问题**：求解 $s_k = \arg\min_{s \in \mathcal{D}} \nabla f(x_k)^T s$
2. **步长计算**：$\gamma_k = \frac{2}{k+2}$（或线搜索）
3. **更新**：$x_{k+1} = (1 - \gamma_k) x_k + \gamma_k s_k$

### 6.2 与投影梯度法的对比

**Frank-Wolfe的优势**：
- **投影自由**：只需解决线性最小化，而非投影到复杂集合
- 当 $\mathcal{D}$ 是多面体时，线性最小化可能比投影更高效
- 稀疏解：解天然是原子的凸组合

**Frank-Wolfe的劣势**：
- 收敛速度较慢：$O(1/k)$ 而非 $O(1/k^2)$
- 对于非多面体约束集合，线性最小化可能仍困难

> [!example] SVM与Frank-Wolfe
> 支持向量机的对偶问题可以用Frank-Wolfe求解。线性最小化子问题退化为寻找支持向量，复杂度与活跃约束数成正比。

### 6.3 收敛性分析

对于强凸目标函数 $f$，Frank-Wolfe达到 $O(1/k)$ 的收敛率：

$$f(x_k) - f(x^*) \leq \frac{2C_f}{k+2}$$

其中 $C_f$ 是与函数曲率相关的常数。

**条件梯度滑动**（Conditional Gradient Sliding）结合了Frank-Wolfe与Nesterov加速的思想，可以达到更好的收敛率。
